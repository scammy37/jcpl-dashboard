💬 Option A — Use Claude (Recommended)

Open a Claude conversation, attach your JCP&L bill PDF, and paste this prompt:

Extract the billing data from this JCP&L electricity bill and return a single JSON object in this exact format — no markdown, no explanation: { "label": "Mon YY", "period": "Mon DD–Mon DD, YYYY", "days": 30, "kwh": 1414, "cost": 282.62, "temp": 72, "rate": "Standard", "onPeak": null, "offPeak": null, "onPct": null, "offPct": null } Rules: - label = end month of billing period, 2-digit year (e.g. "Jun 26") - days = integer - kwh = integer - cost = number, 2 decimal places (electricity charges only, exclude budget billing adjustments) - rate = "Standard" or "Time-of-Day" - If Time-of-Day: fill in onPeak, offPeak, onPct, offPct — otherwise set all four to null - temp = average temp °F if shown on the bill, otherwise null
Then update the dashboard:
1
Copy the JSON entry Claude returns
2
Open data.json in VS Code or any text editor
3
Add a comma after the last } on the final entry, then paste the new entry on a new line before the closing ]
4
Save, then run: git add data.json && git commit -m "Add [Month] bill" && git push
5
Dashboard updates immediately — refresh the page

⚡ Option C — Run the Script (Fastest After Setup)

add_bill.py is in your repo. Get the JSON from Claude (Option A), paste it into the script, and it updates data.json, commits, and pushes automatically — no editing files manually.

Each new bill:
1
Use the Option A prompt with Claude at claude.ai to get the JSON entry for your bill
2
Open PowerShell in Documents\jcpl-dashboard (right-click folder → Open in Terminal)
3
Run the script — no arguments needed:
python add_bill.py
Then paste the JSON from Claude and press Enter twice
4
Review the values, press Y to commit and push — done
💡 You can also pass the JSON directly as an argument:
python add_bill.py '{"label":"Jun 26","period":"Jun 04–Jul 03, 2026",...}'

📋 data.json Entry Format

Each bill is one line. Your most recent entry for reference:

{"label":"May 26","period":"May 05–Jun 03, 2026","days":30,"kwh":1717,"cost":312.30,"temp":66,"rate":"Time-of-Day","onPeak":331,"offPeak":1386,"onPct":19.3,"offPct":80.7}
💡 temp: If your bill doesn't show average temperature, use the monthly average for Rockaway, NJ and fill it in manually.

💡 cost: Use electricity charges only — exclude any budget billing adjustments or credits.
